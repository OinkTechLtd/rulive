# 📺 IPTVPlayer LIVE APP — Современный IPTV плеер 2026

Современный IPTV-плеер с поддержкой M3U/M3U8/HLS/DASH, работающий прямо в браузере.

## 🔐 Безопасность

Приложение включает проверку безопасности (капча) для защиты от ботов. При первом посещении вам потребуется пройти проверку.

## Файлы проекта

```
IPTVPlayer-LIVE/
├── verify.html       ← Проверка безопасности (капча)
├── index.html        ← Лендинг с туториалом
├── player.html       ← Основной плеер с выбором из 3 плееров
├── embed.html        ← Embed плеер (для iframe)
├── css/
│   ├── landing.css
│   └── player.css
└── js/
    └── player.js
```

## Как использовать

1. Открой `verify.html` — пройди проверку безопасности
2. После проверки попадёшь на `index.html` — лендинг
3. Нажми «Открыть плеер» → `player.html`
4. Вставь M3U ссылку
5. Выбери группу каналов из выпадающего списка
6. Кликни на канал и смотри!

## 🔴 Три плеера в одном

В приложении реализовано **3 плеера** для максимальной совместимости:

### Плеер 1: Основной (player.js)
- Использует встроенный player.js из репозитория
- Поддержка HLS, DASH, MP4, WebM, TS
- Выбор качества, PiP, полноэкранный режим

### Плеер 2: Резервный CDN
- Три источника для отказоустойчивости:
  - `https://cdn.jsdelivr.net/gh/OinkTechLLC/cdnplayerjs@main/playerjs.js`
  - `https://cdn.jsdelivr.net/gh/OinkTechLtd/cdnplayerjs@main/playerjs.js`
  - `https://cdn.jsdelivr.net/gh/twixoffltdco/cdnplayerjs@main/playerjs.js`
- Автоматическое переключение если один не работает

### Плеер 3: VideoCDNHub
- Использует `https://videocdnhub.tatnet.app/?src=URL_ПОТОКА`
- Альтернативный способ воспроизведения

**Выбор плеера:** Нажмите кнопку 🎬 в верхней панели плеера для выбора нужного варианта.

## Поддерживаемые форматы

- M3U / M3U8
- HLS (HTTP Live Streaming)
- MPEG-DASH
- MP4, WebM, TS, FLV

## Embed код

1. Выберите канал в плеере
2. Нажмите кнопку 🔗 (Embed)
3. Скопируйте iframe код
4. Вставьте на свой сайт — откроется именно выбранный канал
